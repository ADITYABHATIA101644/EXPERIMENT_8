**# Secure MERN Auth & RBAC System
**Experiment 3.1.1, 3.1.2, & 3.1.3 Combined**

This project is a full-stack MERN application that demonstrates a secure authentication flow using JWT and Role-Based Access Control (RBAC).

## 🚀 Live Demo
https://experiment-8-ggkv.onrender.com

## 🛠️ Features
* **Experiment 3.1.1:** Login form with React State Management (React Hook Form) and Material UI.
* **Experiment 3.1.2:** Protected Routes using JWT (JSON Web Tokens) stored in LocalStorage.
* **Experiment 3.1.3:** Role-Based Access Control (RBAC) with an Admin-only Dashboard and restricted API access.

## 💻 Tech Stack
- **Frontend:** React 18, Material UI, React Router, Axios.
- **Backend:** Node.js, Express.js.
- **Database:** MongoDB Atlas (Mongoose).
- **Security:** Bcrypt.js (Password Hashing), JWT (Authentication).

## 🔧 Installation & Setup
1. Clone the repo: `git clone <your-repo-link>`
2. Install Server deps: `cd server && npm install`
3. Install Client deps: `cd client && npm install`
4. Create a `.env` in the server folder with your `MONGO_URI` and `JWT_SECRET`.
5. Start the project: `npm run start` (from root).
